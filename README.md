## Downloading

```bash
git clone https://github.com/GoldenManBel/App_NestJS_Microservices.git
```

## Installing NPM modules

```bash
npm install
```

## Running application

```bash
running database postgresql
```

```bash
# watch mode
npm run start:dev 
```
or
```bash
npm run start
```

## Auto-fix and format

```bash
npm run lint
```
